OUR PROJECTS ON ARITIFICIAL INTELLIGENCE

Brief Description of project:

This project is designed to provide virtual assistance to the users in terms of providing all the information and data searched. This project also have features like hearing and speaking. Some modules are also given to this project. It has connectivity with internet so the data from any site can be accessed through this module. This project is mainly based on artificial intelligence with assistant named as ASA. ASA is able to speak as well as hear the queries of users. It also provides the option to search through keyboard. This virtual assistant has its coding in python with many packages and modules.  

Field Of Project:

This project is mainly used to provide assistant to users. This is an project which runs on every platform. This project is free of cost. ASA has good quality of voice which is easily understandable. It has easy access over the websites and accepts the commands easily. This module can be used in phones as well as in pc’s too. All these function of this project makes it eye catchy. Most important function of this module is it can be used by every one easily.   
Use of ASA virtual assistant:
1.Provide information such as weather, facts from e.g. Wikipedia.
2.Can provide the answer of search question by speaking.
3.One can get answer of query without using keyboard.
